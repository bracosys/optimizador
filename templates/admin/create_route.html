    Volver
    Guía para archivos GPX

    Los archivos GPX (GPS Exchange Format) son un formato estándar que contiene datos de rutas GPS. Puedes obtenerlos de diversas formas:

    • Exportando rutas desde aplicaciones como Google Maps, Strava, o MapMyRide
    • Descargando recorridos prefijados de sitios de mapas
    • Grabando tu propio recorrido con un dispositivo GPS o aplicación móvil

    Para mejores resultados:

    • Asegúrate de que el archivo GPX contiene una secuencia coherente de puntos
    • Verifica que el punto de inicio y fin estén claramente definidos
    • Si combinas múltiples archivos, estos se procesarán en el orden en que los subas
